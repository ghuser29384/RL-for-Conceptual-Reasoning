# Metaphilosophy Synthetic Argument Library

This route publishes 1,000 model-authored argument–critique records: 250 positions with four critiques each, across 25 domains.

The release is explicitly **synthetic and unrated**. It is separate from the expert-rated LMCA corpus and must not be counted in LMCA dataset metrics.

## Files

- `index.html` — accessible static shell
- `styles.css` — responsive editorial interface
- `app.mjs` — search, filters, pagination, permalinks, and client-side exports
- `data/positions.json` — generated at build time from the compressed release payload
- `data/critiques.json` — generated at build time from the compressed release payload
- `data/manifest.json` — generated release metadata and checksums
